Описание

ExportProject_OOP — учебный проект, реализующий ключевые концепции объектно-ориентированного программирования на C++.
Включает абстрактный базовый класс, четыре наследника, интерфейсы, перегрузку операторов, обработку исключений, сериализацию и сохранение данных.

Функционал проекта:

  • Абстрактный базовый класс ExportControl
  • Наследники: PipelineExport, MarineExport, RailExport, TruckExport
  • Интерфейсы: IShippingControl, IExportEfficiency
  • STL-контейнеры: std::vector с shared_ptr
  • Перегрузка операторов: ==, <, +, ++
  • Обработка исключений
  • Ввод/вывод данных, сохранение и загрузка из файла
  • UML-диаграмма структуры проекта

Открывается и собирается в Visual Studio 2022+ (или через g++).

Инструкция по запуску

  1. Распакуйте архив
  2. Откройте файл ExportProject_OOP_final_OPERATORS.sln в Visual Studio
  3. Соберите и запустите проект (Debug x64 или Release x64)
  4. Если используете g++:
    g++ -std=c++17 -I. main.cpp ExportControl.cpp MarineExport.cpp RailExport.cpp TruckExport.cpp PipelineExport.cpp -o export_app
